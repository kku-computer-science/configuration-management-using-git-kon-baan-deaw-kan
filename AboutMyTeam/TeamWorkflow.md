ทีมของเราทำงานร่วมกันโดยใช้ Git และ GitHub เพื่อแบ่งหน้าที่และรวมโค้ดอย่างเป็นระบบ โดยมีขั้นตอนดังนี้:

## 1️ แบ่งหน้าที่ตามสมาชิกในทีม
- แต่ละคนรับผิดชอบโค้ดส่วนที่ได้รับมอบหมาย เช่น Quick sort / Bubble sort
- เก็บซอร์สโค้ดไว้ในโฟลเดอร์ `Project`

## 2️ ทำงานบน Branch ส่วนตัว
- สร้าง Branch ชื่อ **ชื่อจริง + รหัส 4 ตัวท้าย** เช่น `Somsri_1234`
- พัฒนาและแก้ไขงานเฉพาะบน Branch ของตนเองเท่านั้น
- ห้ามแก้ไขงานบน `main` โดยตรง

## 3️ Commit และ Push อย่างสม่ำเสมอ
คำสั่งตัวอย่าง:
```
git status
git add .
git commit -m "Update sorting logic"
git push origin BranchName
```

## 4️ รวมงานเข้ากับ Main Branch
- เปิด Pull Request (PR) เมื่อมั่นใจว่างานเสร็จและถูกต้อง
- ให้เพื่อนในทีมตรวจสอบ (Review) ก่อน Merge
- หากมี Conflict ต้องช่วยกันแก้ไขก่อน Merge

### เป้าหมายของการทำงานร่วมกัน
- ทำงานทีมเป็นระบบ
- ลดข้อผิดพลาดตอนรวมงาน
- ตรวจสอบและติดตามงานของแต่ละคนได้ง่าย
